# A1 Audit — Admin Realtime Source Policy Wave

## Verdict

Wave is acceptable and bounded.

## What was proved

- operator override is persisted outside static files
- effective realtime policy is exposed explicitly as `static`, `override`, and `effective`
- realtime runtime consumers read the effective policy layer instead of static config only
- admin shell can apply and reset the narrow override without becoming a generic config editor
- frontend handoff contract now states restart semantics honestly

## Remaining constraints

- restart is still required before runtime source-selection changes fully apply
- override does not extend to backfill, validation, raw-tx, URLs, secrets, or capability matrices
- this wave does not attempt hot-reload orchestration or generic provider management
